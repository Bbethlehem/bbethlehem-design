# Agentes Especializados — Bakery Bethlehem LLC

Roles on-demand adaptados del repo [agency-agents](https://github.com/msitarzewski/agency-agents) (MIT).

## Cómo usarlos

En Claude Code o Claude Desktop, pide:
> "Actúa como el agente instagram-bakery y genera un calendario de contenido para mayo"

O en Claude Code, referencia el archivo:
> "Lee agents/instagram-bakery.md y genera 5 ideas de posts para esta semana"

## Agentes disponibles

| Agente | Archivo | Uso principal |
|--------|---------|---------------|
| Instagram Bakery | `instagram-bakery.md` | Posts, Reels, Stories, carruseles |
| SEO Local | `seo-local-bakery.md` | Google Reviews, SEO local, Maps |
| Content Creator | `content-creator-bakery.md` | Blog, email, contenido multicanal |
| Social Media | `social-media-bakery.md` | Estrategia multi-plataforma |
| Growth Bakery | `growth-bakery.md` | Crecimiento, retención, referidos |
